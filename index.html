Values Buttons for Timeles

The ideas is to pass in an array of objects containing a value and a display string.

const data = [
  { display: "0:05", value: 300 },
  { display: "0:10", value: 600 },
  { display: "0:30", value: 1800 },
  { display: "1:00", value: 3600 },
  { display: "2:00", value: 7200 },
  { display: "3:00", value: 10800 },
];

Then create a list of buttons for each object in the array.
